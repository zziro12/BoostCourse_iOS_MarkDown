# MarkDown 작성법 공부!
## 1.MarkDown이란 무엇인가
MarkDown은 텍스트 기반 웹상에서 글을 쓰는 모든 사람들을 위한 글쓰기 도구(서식, 포맷, 양식) 입니다.   
GitHub를 통해 각광받기 시작했고 MD을 통해 소스설명, 프로젝트단계별 정리 등을 간단하고 빠르게 할 수 있다   
## 2.MarkDown 문법
### 2.1 줄바꿈
문장 끝에 3칸이상 띄어쓰기를 하면 줄이 바뀐다 "   "   
~~~
문장을 마치고.
띄어쓰기 없이 그냥 쓴다면 개행이 되지않고(   )   
뒤에 띄어쓰기를 3칸이상 해주면 개행되는것을 볼수있다.
~~~
문장을 마치고.
띄어쓰기 없이 그냥 쓴다면 개행이 되지않고(   )   
뒤에 띄어쓰기를 3칸이상 해주면 개행되는것을 볼수있다.   
### 2.2 Header(헤더)
~~~
# 헤더 h1
## 헤더 h2
### 헤더 h3
#### 헤더 h4
##### 헤더 h5
###### 헤더 h6
~~~
# 헤더 h1
## 헤더 h2
### 헤더 h3
#### 헤더 h4
##### 헤더 h5
###### 헤더 h6
1~6 까지 지원한다.
